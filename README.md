# ast-view

View the ast of JavaScript code in realtime.

Check out http://ast.evanlucas.com to see it in action.

I got sick of having to do:

```js
console.log(require('util').inspect(obj, {
  colors: true
, depth: null
}))
```

## Install

```bash
$ npm install ast-view
```
